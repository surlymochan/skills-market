# Skills Market 索引

本 skill 提供三方 Agent Skills 综合市场的**源级索引**，便于发现各明星产地及安装方式。

## 何时使用

- 需要查找或推荐「哪里可以找到更多 Agent/Claude Skills」时。
- 需要给出某类技能（如科学、n8n、官方）的入口或安装命令时。

## 索引与防膨胀

- 本仓库仅维护**来源级**列表（每个产地一行），不在此枚举具体 skill 名称，避免上下文膨胀。
- 详细列表与检索请使用本仓库中的脚本：`scripts/skills-market.sh list` / `scripts/skills-market.sh search <关键词>`，或直接打开下方 references 中的市场文档。
